Use this page when you want the fastest path to a working setup.

## Reuse these values everywhere

```bash
export APPLEAI_BASE_URL=https://api.appleai.codes/v1
export APPLEAI_API_KEY=YOUR_APPLEAI_API_KEY
export APPLEAI_MODEL=anthropic/claude-sonnet-4
```

## Basic flow

1. Create or copy an `appleai.codes` API key.
2. Pick a routed model path such as `anthropic/claude-sonnet-4` or `openai/gpt-5`.
3. Open the guide for your tool and paste the shared values into its config.
4. Run one simple request to confirm the endpoint and key work together.

## Verify the connection first

```bash
curl https://api.appleai.codes/v1/models \
  -H "Authorization: Bearer $APPLEAI_API_KEY"
```

If that call succeeds, your key is valid and the base URL is reachable.

## Recommended order

1. Start with **Claude Code**, **Codex CLI**, or **OpenCode**. They expose direct config files.
2. Move to **Cursor**, **Cline**, **Continue**, or **Roo** when you want editor integration.
3. Use the **MITM-only tools** flow only when the product does not let you enter a custom base URL.
