`appleai.codes` gives you one endpoint for your coding tools. These guides show you how to connect Claude Code, Codex CLI, Cursor, OpenCode, Continue, Roo, and other OpenAI-compatible clients without repeating the same setup work in every app.

Base URL for all tools: `https://api.appleai.codes/v1`
